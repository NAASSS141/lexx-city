# V22 人物交互系统 QA

本阶段新增人物交互系统，不修改原有功能、剧情、证据、存档、结局逻辑。

## 新增能力

- `characterInteractionProfiles`：主要角色基础互动内容
- `characterEvidenceResponses`：角色对关键证据的反应
- `showCharacterInteraction()`：点击角色打开人物交互菜单
- `showCharacterEvidenceMenu()`：选择已有证据并向角色出示
- `speakCharacterInteraction()`：将人物交互转化为对白、情绪和镜头反馈

## 支持互动

- 观察：显示角色当前状态和细节描写
- 追问：触发短对白
- 出示证据：从已有证据中选择一条进行反应
- 请求提示：根据角色和当前任务给出下一步方向
