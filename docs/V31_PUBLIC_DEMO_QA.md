# V31 Public Demo QA

## 版本目标

V31 在 V30 Release Candidate 基础上，面向公开试玩做 polish。目标不是增加新剧情或大系统，而是优化“首次打开、开始试玩、试玩结束、反馈收集”的体验，让 Demo 更适合发给朋友、测试玩家或潜在合作者。

---

## 本版优化内容

### 1. 首页试玩说明

首页新增 `PUBLIC DEMO` 说明卡，明确：

- 建议优先体验第一案
- 预计试玩时长：25–40 分钟
- 推荐浏览器：Chrome / Edge / Safari 最新版
- 玩法重点：点热点、问人物、收证据、回律所推理、进法庭翻盘

### 2. 新增试玩说明入口

首页资料与系统区新增按钮：

- `试玩说明`
- `已知问题`
- `反馈模板`

对应函数：

- `showPublicDemoGuide()`
- `showKnownIssues()`
- `showPublicDemoFeedback()`

### 3. 新增本地反馈模板

新增纯静态反馈模板，不接入真实联网表单。

反馈模板包含：

- 试玩设备
- 浏览器
- 是否开启声音
- 试玩到哪里
- 是否完成第一案
- 最喜欢的部分
- 最不清楚的操作
- 是否卡住
- 手机端是否好点
- 画面 / 文字是否清楚
- 建议改进

玩家可以点击“复制模板”，复制后发给作者。

### 4. 试玩结束反馈提示

第一案真相结局后新增反馈提示卡，鼓励试玩者记录体验。  
终局结局也加入反馈提示，但不影响原有结局逻辑。

### 5. 已知问题 / 注意事项

新增本地说明弹窗，提示：

- 浏览器音频自动播放限制
- 手机端极窄屏可能裁切宽幅 CG 边缘
- 首次打开需要加载高质量图像
- 存档保存在当前浏览器本地
- 建议优先试玩第一案
