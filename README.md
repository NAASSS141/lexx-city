# 法域之城：雨夜证词 V54 Final Public Demo

一个可直接部署到 GitHub Pages 的静态网页独立游戏 Demo。体验目标是 **轻探索 + 视觉小说 + 分镜叙事 + 场景交互 + 法律推理**。

V54 是最终公开试玩准备版，重点整理了首次打开、开始试玩、试玩结束和反馈收集体验。

## 推荐试玩方式

- 推荐顺序：第一案 → 第二案 → 第三案
- 第一案预计时长：25–40 分钟
- 全流程预计时长：60–90 分钟
- 推荐浏览器：Chrome / Edge / Safari 最新版
- 推荐设备：桌面端最佳，手机端可玩
- 声音建议：建议开启声音；浏览器可能需要首次点击后才允许播放
- 存档说明：存档保存在当前浏览器本地，清理站点数据会删除存档

## 玩法说明

1. 点击“新游戏”进入第一案。
2. 点击场景中的热点调查现场。
3. 点击人物进行观察、追问、出示证据或请求提示。
4. 获得证据后进入证据包和资料馆查看。
5. 回到律所办公室，在证据墙组合推理。
6. 完成轻解谜后进入法庭，提交关键证据。
7. 第一案结束后进入第二案，第二案结束后进入第三案。
8. 终局后复制反馈模板，记录卡点和建议。

## 反馈建议

本项目保持纯静态网页，不接入真实联网表单。试玩后请在游戏首页点击“反馈模板”，复制后发给作者。

建议重点反馈：

- 玩到哪里卡住了
- 哪个按钮或热点不好点
- 哪段任务提示不清楚
- 哪个证据、推理或法庭条件不明白
- 手机端是否有裁切、看不清或误触
- 哪个画面、人物或证据最有记忆点
- 最希望下一版优化什么

## 部署方式

1. 解压 zip。
2. 确认 `index.html` 在仓库根目录。
3. 将解压后的全部文件上传到 GitHub 仓库根目录。
4. 打开仓库 `Settings -> Pages`。
5. Source 选择 `Deploy from a branch`。
6. Branch 选择 `main`。
7. Folder 选择 `/root`。
8. 保存并等待 GitHub Pages 生成网址。

正确结构：

```text
你的仓库/
├── index.html
├── style.css
├── game.js
├── data/
├── assets/
├── docs/
├── README.md
└── GITHUB_PAGES_DEPLOY.md
```
