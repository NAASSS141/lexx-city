# V17 轻解谜互动系统 QA

本阶段新增轻解谜互动系统，不修改原有剧情、证据、推理组合、结局和存档结构。

## 新增玩法

- 证据排序：把线索按时间或调查路径排序
- 证词矛盾：从证词中指出最值得追问的矛盾句
- 提交证据：选择正确证据反驳错误叙事

## 新增入口

- 首页：轻解谜
- 顶部 HUD：解谜

## 数据与存档

- 新增 `state.puzzles`
- 旧存档会自动补齐 `puzzles: []`
- 解谜完成只增加轻量真相值，不替代原有证据组合逻辑
- 不影响结局判定
