## 部署到 GitHub Pages

1. 解压本 zip
2. 上传解压后的全部文件到 GitHub 仓库根目录
3. 打开 Settings -> Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，目录选择 `/root`
6. 保存后等待 GitHub 自动生成网址
