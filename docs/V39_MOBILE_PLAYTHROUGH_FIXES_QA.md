# V39 电脑端（兼容命名）真实试玩专项修复 QA

## 版本目标

V39 基于 V38 第一案全流程真人试玩修复版，专门修复电脑端（兼容命名）真实试玩反馈。目标不是新增系统，而是让电脑端（兼容命名）体验更顺：少滑动、好点击、弹窗正确覆盖、文字可读、CG 和主画面不明显裁切。

---

## 收到的问题

### 1. 场景描述存在不必要的感悟式文案

反馈示例：

- `雨越下越密。你抵达商业区路口，案件从这里重新呼吸。`
- `便利店灯还亮着，货架之间藏着比雨声更碎的呼吸。`

要求：描述场景即可，不需要加入体会、感悟或替玩家解释气氛。

### 2. 电脑端（兼容命名）页面仍需要上下滑动

反馈：人物旁白、对话等内容需要滑动页面才能看到，很不方便。

### 3. 第二案、第三案质量需要跟上

反馈：第一案内容逻辑设置质量不错，第二第三案质量也要提上来。

---

## 逐条修复项

### 1. 文案清理

已替换或删除：

- `雨越下越密。你抵达商业区路口，案件从这里重新呼吸。`
  - 改为：`雨越下越密，你抵达商业区路口。`
- `便利店灯还亮着，货架之间藏着比雨声更碎的呼吸。`
  - 改为：`便利店灯还亮着。`
- `车牌残片开始有了骨头`
  - 改为：`车牌碎片编号只剩后三位：7K2。`
- `利息像藤蔓一样爬满每页纸`
  - 改为：`保险柜里有一份旧合同。`
- 法庭观察、天台观察中类似“像一份迟到的判词”等感悟式文案
  - 改为中性记录句

同时复查并清除以下明显问题表达：

- `雨越下越密。你抵达商业区路口，案件从这里重新呼吸。`
- `便利店灯还亮着，货架之间藏着比雨声更碎的呼吸。`
- `重新呼吸`
- `比雨声更碎`
- `像藤蔓`
- `证人席空着，却像`
- `像一份迟到的判词`
- `真正意义`

### 2. 电脑端（兼容命名）减少上下滑动

新增 V39 桌面布局规则：

- `#gameScreen.screen.active` 改为 100vh 内部布局
- 电脑端（兼容命名）页面整体 `overflow:hidden`
- 顶部栏进一步压缩
- 主场景区高度改为 `clamp(260px, 38vh, 365px)`
- 右侧栏宽度压缩到 322px
- 对话框压缩到 `max-height:26vh`
- 人物头像高度压缩到 104px
- 对话文本区内部滚动，避免整个页面滚动
- 行动按钮区和证据包改为内部滚动
- 低高度屏幕增加额外压缩规则

### 3. 保留弹窗 fixed 修复

继承 V38 修复：

- `.modal` 继续强制 `position: fixed`
- 系统面板继续覆盖显示
- 打开弹窗锁定页面滚动
- 地图、图鉴、时间线、任务、存档、设置等不再掉到页面底部

### 4. 第二、第三案质量整理

不新增大型资源、不修改核心逻辑的前提下，做了清晰化整理：

- 第二案目标从“氛围描述”改为“医院账本与旧码头委托链”
- 第三案目标从“匿名函意义”改为“灰塔资本与终局法庭”
- 第二案组合推理改为：
  - `核对医院账本来源`
  - `核对旧码头委托链`
- 第三案组合推理改为：
  - `核对灰塔资本线索`
  - `核对天台会面线索`
- 第三案法庭行动文案改为 `提交匿名函证据`
- 医院、码头、档案中心、天台、终审法庭的场景描述进一步明确证据对象

说明：本轮没有新增第二、第三案 CG 或剧情大段重写，避免引入大型资源和逻辑风险。后续可单独做 V40/V41 来追平第二案和第三案演出质量。
