# 法域迷城：第一案《算法之下》

这是一个可直接运行的 React + TypeScript + Vite 网页剧情法律游戏项目。

## 本地运行

```bash
npm install
npm run dev
```

打开终端显示的本地地址即可体验游戏。

## 生产构建

```bash
npm run build
npm run preview
```

构建产物在 `dist/` 目录，可部署到 Vercel、Netlify、腾讯云静态网站托管、阿里云 OSS + CDN 等静态托管服务。

## 已实现内容

- 首页、城市地图、剧情对话、场景调查、证据拼接、庭审、结局、知识库、案件档案
- 主角为崔知微，NPC 统一称呼“崔律”
- 首发案件《算法之下》完整可通关
- 4 个结局：S、A、B、C
- localStorage 本地存档，刷新后可继续
- 不依赖外部图片资源，全部使用 CSS、Emoji、卡片和渐变视觉
- 移动端与桌面端响应式适配
- 页面底部包含法律声明
