# V12 功能说明

## 证据拖拽推理棋盘

V12 新增“推理棋盘”：

- 可将已获得证据拖入 3 个推理槽
- 移动端可以点击证据自动填入空槽
- 三条证据匹配某条组合推理时，可直接完成推理
- 保留原来的传统“证据组合”面板

这个系统使用 `combos[].req` 作为判定源，因此后续新增组合推理时，推理棋盘会自动读取数据，无需额外写 UI。

## 语音演出提示音

V12 新增“角色语气提示音”：

- 每个角色发言时播放短促音色
- 不是完整真人配音，因此体积小、部署简单
- 可在“语音演出”面板中开启/关闭、调节音量、试听角色音色

音频文件位于：

```text
assets/audio/voice_*.wav
```

如需替换真人语音或更精致的角色音色，可修改：

```text
data/game-data.json
data/game-data.js
metadata.voiceFiles
```
