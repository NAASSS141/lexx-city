# V19 镜头语言系统 QA

本阶段新增镜头语言系统，并保留 V18 人物舞台能力。

## 新增能力

- `cameraDirector`：场景镜头控制层
- `cameraCaption`：镜头状态提示
- `setSceneCamera()`：根据当前场景应用基础镜头
- `pulseCamera()`：临时镜头变化，用于热点、证据、关键台词、转场
- `applyDialogueCamera()`：根据当前发言人和场景自动切换近景 / 对峙 / 冲击镜头

## 支持镜头

- wide：远景，展示场景整体
- medium：中景，适合人物对话
- close：近景，强调当前发言角色
- insert：物件插入镜头，强调证据或热点
- impact：冲击镜头，强调关键台词或推理成立
- black：黑场字幕，适合章节转场
- split：左右对峙镜头，适合法庭和对峙场景

## 保留项

- 不删除原有对话框
- 不删除热点探索
- 不删除人物舞台
- 不删除行动按钮
- 不删除任务系统
- 不删除轻解谜系统
- 不影响音频、分镜、证据、结局、存档逻辑
