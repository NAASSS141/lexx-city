# V16 章节推进与任务系统 QA

本阶段新增章节任务系统，不修改剧情数据、证据逻辑、结局逻辑和存档结构。

## 新增能力

- `missionDefs`：案件阶段目标定义
- `renderTaskTracker()`：侧边栏任务追踪
- `evaluateTaskProgress()`：阶段完成检测
- `showMissionPanel()`：完整任务档案面板

## 覆盖案件

- 第一案：雨夜证词
- 第二案：沉默账本
- 第三案：灰塔匿名函

每案 5 个阶段，按照“调查 → 证人/证据 → 推理链 → 庭审/终局”的关卡结构展示。

## 保留项

- 不删除原有行动按钮
- 不修改证据获取逻辑
- 不修改组合推理判定
- 不修改结局判定
- 不修改存档结构
- 任务进度完全基于现有 state 推导
